# Linux面试题

## 1.文件和目录操作命令

+ ls 列出文件目录或内容
+ cd 改变目录
+ mkdir 新建文件夹
+ mv 移动
+ cp 复制
+ rm 删除 

【-r】 递归文件夹下的选项

## 2. 查找文件命令

atai@atai-virtual-machine:~$ find ~/ -name "1.txt"
/home/atai/1.txt

grep 是在文本下查找

## 3. 查看文件内容

cat 

tail 查看尾部

## 4. 进程管理

ps -ef 查看进程信息

kill -9 强制杀死

不加-9 让自己杀死

## 5.软件包处理

apt-get是ubuntu的下载命令

## 6. 查看ip

ip addr show
