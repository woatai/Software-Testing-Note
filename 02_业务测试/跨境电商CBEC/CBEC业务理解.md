# CBEC 业务理解

## 多币种、汇率、支付测试点

题目：

> 一个跨境电商下单流程涉及「多币种 + 汇率 + 支付」，你会重点关注哪些测试点？

考点：

- 业务敏感性
- 金额 / 精度 / 时区

解答：

> 1. 查看多币种在订单信息的显示是否正确
> 2. 汇率计算精度是否正确
> 3. 汇率变更后对历史订单是否有影响
> 4. 不同时区下单时间是否正确
> 5. 支付成功，订单状态是否显示正确
> 6. 支付失败时，库存是否回滚
> 7. 网络异常 / 重复点击

## 跨境电商与普通电商的测试差异

**跨境电商系统与普通电商相比，测试重点有哪些不同？**

> + 多币种&汇率
>   + 不同国家的币种显示正确
>   + 汇率计算在商品列表、详情页、订单页正确
>   + 下单、支付、退款能正确换算对应的汇率
>   + **汇率发生变化不影响历史订单**
> + 税收情况
>   + 能够按照对应的收货地址显示对应税收情况
>   + 能够在商品列表、详情页、下单页显示正确
> + 时区的显示
>   + 能按照用户所在地区显示正确的时间
> + 支付方式
>   + 不同国家支持不同的支付方式
>   + 跨境支付风控校验
> + 物流国内外的问题
>   + 不同国家显示可选的物流方式不同
>   + **清关状态、异常状态是否能正确展示**
>   + **国内段 + 国际段物流**是否能衔接

## 跨境电商下单流程易出问题环节

**跨境电商下单流程中，哪些环节最容易出问题？为什么？**

> + 汇率计算
>   + 多个国家不同的汇率计算
>   + 汇率实时计算的问题
> + 支付到账问题
>   + 支付链路长
>   + 第三方参与
> + 退款问题
>   + 原币种和实时汇率
>   + 可能多次退款
>   + 跨境周期长
> + 物流问题
>   + 国内 + 国外衔接段物流问题
>   + 清关流程复杂

## 多币种、多语言测试点

**多币种、多语言场景下，你会重点关注哪些测试点？**

解答：

> + 多语言
>   + 不同语言切换后，页面文案是否完整、准确
>   + 是否存在中英文混用、乱码、截断问题
>   + 语言切换是否影响下单、支付页面
> + 多币种
>   + 在商品页、列表页、详情页、后台显示正确
>   + 币种符号、位置是否符合当地习惯
>   + 币种切换后是否影响历史订单显示
> + 汇率
>   + 汇率换算精度是否符合业务规则
>   + 不同环节（下单、支付、退款）是否使用一致的汇率
>   + 多次换算是否产生累计误差
>   + 汇率变动是否影响已生成订单
