# 音频转文字使用说明

## 作用

把面试录音 `m4a / mp4a / mp3 / wav` 转成带时间戳的文字稿。

默认输出：

- 简体转写：`音频名.转写.简体.txt`

## 使用命令

在 PowerShell 7 或普通 PowerShell 中执行：

```powershell
cd E:\Note\01_origin
.\03_测试求职\音频转文字\音频转文字.ps1 -AudioPath "E:\Note\01_origin\03_测试求职\云起面试准备客户运营\开创大道.m4a"
```

## 可选参数

使用更快但准确率低一点的模型：

```powershell
.\03_测试求职\音频转文字\音频转文字.ps1 -AudioPath "音频路径.m4a" -Model base
```

保留中间 wav 文件：

```powershell
.\03_测试求职\音频转文字\音频转文字.ps1 -AudioPath "音频路径.m4a" -KeepWav
```

## 说明

- 默认使用 `small` 模型，准确率比 `base` 更好，但会慢一点。
- 脚本内部会先生成原始转写文本，再转成简体，但最终只保留简体结果。
- 转写结果可能会有错别字，专业名词、人名、公司名需要人工校对。
- 当前脚本依赖本机 Codex Python 环境里的 `pywhispercpp`、`imageio-ffmpeg`、`opencc-python-reimplemented`。
- 如果换电脑或环境丢失，需要先安装依赖：

```powershell
C:\Users\atai\.cache\codex-runtimes\codex-primary-runtime\dependencies\python\python.exe -m pip install pywhispercpp imageio-ffmpeg opencc-python-reimplemented
```
