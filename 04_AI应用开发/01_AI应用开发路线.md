# AI 应用开发阶段学习路线

## 01_AI客服最小闭环

### 阶段目标

做出普通 AI 客服 `/chat` 闭环。

流程：

```text
前端输入问题
↓
后端接收请求
↓
后端调用大模型
↓
返回 AI 客服回复
↓
页面展示回复
```

### 学习顺序

```text
Python 项目结构
↓
HTTP 接口
↓
FastAPI
↓
Pydantic
↓
LLM API 调用
↓
Prompt 工程
```

### 阶段产出

能运行 `ai-support` 项目。

用户在前端输入问题后，前端调用后端接口，后端调用大模型生成回复，页面展示 AI 客服回复。

### 验收标准

1. 能解释 `ai-support` 项目里每个核心文件的职责。
2. 能测试 `/chat` 接口。
3. 能看懂 `/chat` 的成功响应。
4. 能看懂 `/chat` 的错误响应。
