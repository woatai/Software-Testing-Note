# Prompt 与结构化输出

Prompt 不是写漂亮的话，而是把任务边界说清楚，让模型稳定地产出可用结果。

## Prompt 基本结构

- 角色：你要模型以什么身份工作。
- 任务：明确要完成什么。
- 输入：给模型哪些材料。
- 输出：规定格式、字段和层级。
- 约束：不要编造、不要泄露隐私、不会就说明不确定。

## 结构化输出示例

适合 AI 应用的输出格式通常包括：

- Markdown：适合报告、复习计划、面试反馈。
- JSON：适合后端继续处理或前端展示。
- 表格：适合对比岗位要求、技能差距、复习优先级。

## 面试分析助手里的输出

建议固定为四块：

1. 面试问题清单。
2. 回答薄弱点。
3. 推荐复习主题。
4. 可直接背诵的优化回答。

这样后续无论接页面、保存文件，还是继续追问，都比较稳定。
