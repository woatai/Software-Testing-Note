# LLM API 调用

LLM API 调用的核心是：把用户输入组织成模型能理解的上下文，再把模型输出整理成应用可用的结果。

## 基本流程

1. 准备输入：用户问题、文档内容、任务说明。
2. 设计 Prompt：说明角色、目标、输出格式和限制。
3. 调用模型：发送请求并设置模型、温度、最大输出等参数。
4. 解析结果：取出文本或结构化 JSON。
5. 兜底处理：失败重试、格式修复、错误提示。

## 常见应用

- 总结：把长文本压缩成重点。
- 分类：判断问题属于哪类知识点。
- 抽取：从文本中提取问题、时间、岗位、技能点。
- 改写：把回答优化成更适合面试表达的版本。
- 生成：根据输入生成测试点、复习计划或接口用例草稿。
