# Agent 与工具调用

Agent 可以理解为：模型不只是回答问题，还能选择工具、调用工具、读取结果，再继续完成任务。

## 工具调用例子

- 读取本地转写文本。
- 搜索知识库里的相关笔记。
- 生成复习清单。
- 调用接口保存分析结果。
- 根据模板生成 Markdown 报告。

## 和普通 LLM 调用的区别

- 普通调用：输入一段文本，输出一段文本。
- 工具调用：模型判断需要什么工具，再把工具结果纳入下一步推理。

## 适合做进项目里的能力

后续确定练习项目后，可以逐步加入这些能力：

1. 读取转写文件。
2. 分析薄弱点。
3. 自动搜索知识库相关笔记。
4. 生成复习任务。
5. 输出一份可保存的面试复盘报告。
